NexMeet 🚀

A modern full-stack real-time communication platform built for seamless collaboration, featuring high-quality video calling, instant messaging, screen sharing, and desktop sharing.

📂 GitHub Repository:
https://github.com/hs-techlabs/NexMeet-video-call

✨ Features:
Real-Time Video Calling , Instant Chat Messaging  , Screen Sharing , Desktop Sharing , Fast & Responsive UI , Secure Peer-to-Peer Communication , Cross-Platform Accessibility , Real-Time Socket Connection.

🛠️ Tech Stack: 
Frontend: React.js , Tailwind CSS , Socket.IO Client.

Backend: Node.js , Express.js , Socket.IO , WebRTC.

Deployment: Render.

Installation & Setup:

Clone the Repository: git clone https://github.com/hs-techlabs/NexMeet-video-call.git

Navigate to Project Directory: cd NexMeet-video-call

Install Dependencies: npm install

Run Development Server:  npm start
